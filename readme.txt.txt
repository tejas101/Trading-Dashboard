1)Install Xampp - https://www.apachefriends.org/download.html

2) Copy the Trading folder in Xampp's htdocs folder.

3)Start the Xampp and MYSQL server from the control panel.

4)Into PHPMYadmin, import the  sql file present in the Trading folder.

5) Access the index page at http://localhost/Trading/ 
and Statistics page at http://localhost/Trading/Statistics.php
