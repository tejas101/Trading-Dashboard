Question #4

You are supplied trade data in the following format:

trade_id, timestamp, symbol, quantity, price, is_buy, trader

Each line in the data represents a trade.

The task is to accomplish the following:
  1. Render the data in a table.
  2. Allow the user to sort the table by any of the fields.
  3. For each trader, show statistics:
    • Total # of trades
    • Total position per symbol (how many shares of each stock does the trader have).
      • You can assume a position of zero for each symbol at the start of the day.
    • Average buy price per symbol
    • Average sell price per symbol
  4. Show the trades for each symbol in a chart.
    • Either show all symbols in a single chart, or allow the user to select which symbol to view
  5. Bonus: Allow the user to edit the trader field.
    • For example, reassign a trade from LISA to RACHEL.
    • This change should be reflected in the statistics.
For your solution, you are free to use whatever libraries or APIs you see fit.

Please place your solution (along with any additional files/folders you need) in a .zip file (YOURNAME.zip, for example), and attach it in your emailed response.
